# Gobuster Security Considerations

- Restrict execution to approved tenant and operator scopes.
- Run only against targets with explicit authorization.
- Prevent target URL and wordlist paths from exposing sensitive internal data.
- Preserve immutable audit evidence for execution intent and wrapper results.
