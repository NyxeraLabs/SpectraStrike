# Responder Security Considerations

- Restrict execution to approved tenant and operator scopes.
- Limit live responder runs to controlled lab networks and approved interfaces.
- Avoid forwarding credential material or hashes into telemetry output.
- Preserve immutable audit evidence for execution intent and wrapper results.
