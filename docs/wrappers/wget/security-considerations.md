# Wget Security Considerations

- Restrict execution to approved tenant and operator scopes.
- Limit network transfer tests to authorized endpoints.
- Prevent credential material from being passed in cleartext CLI flags.
- Preserve immutable audit evidence for execution intent and wrapper results.
