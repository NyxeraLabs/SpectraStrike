# Netcat Security Considerations

- Restrict execution to approved tenant and operator scopes.
- Limit live sessions to authorized hosts and ports.
- Do not transmit secrets or uncontrolled payloads through ad-hoc command arguments.
- Preserve immutable audit evidence for execution intent and wrapper results.
