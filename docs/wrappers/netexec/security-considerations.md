# NetExec Security Considerations

- Restrict execution to approved tenant and operator scopes.
- Limit live credential use to authorized environments and short-lived test accounts.
- Avoid exposing secrets in command logging and telemetry attributes.
- Preserve immutable audit evidence for execution intent and wrapper results.
