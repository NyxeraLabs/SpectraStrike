# John the Ripper Security Considerations

- Restrict execution to approved tenant and operator scopes.
- Limit live cracking tests to authorized lab data only.
- Avoid telemetry leakage of hashes, candidate secrets, or cracked values.
- Preserve immutable audit evidence for execution intent and wrapper results.
