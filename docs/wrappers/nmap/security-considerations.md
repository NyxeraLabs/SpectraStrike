# Nmap Security Considerations

- Limit execution to approved tenant and operator scopes.
- Protect credentials, tokens, and artifacts from telemetry leakage.
- Run wrapper E2E validation only in controlled infrastructure.
- Record immutable evidence for execution intent and results.
